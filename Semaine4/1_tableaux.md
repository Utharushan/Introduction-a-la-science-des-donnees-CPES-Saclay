Calculez à la main :
- la moyenne de chaque colonne de `T`;
- la moyenne de chaque ligne de `T`;

```{code-cell} ipython3
moy_tot = 0
for i in range(len(T[0,:])):
    moy_col = 0
    for j in range(len(T[:,0])):
        moy_tot += T[j,i]
        moy_col += T[j,i]
    print("Moyenne de la colonne :", moy_col/len(T[:,0]))


moy_ligne = 0
for i in range(len(T[:,0])):
    moy_ligne = 0
    for j in range(len(T[0,:])):
        moy_ligne += T[i,j]
    print("Moyenne de la ligne :", moy_ligne/len(T[0,:]))

print("Moyenne de tous les éléments =", moy_tot/(len(T[:,0])*len(T[0,:])))    
```
